Sistema BCF
Sistema que permite importar desde planillas XLS el control y la generación de archivo XML para el cumplimiento de la Comunicación del Banco Central del Uruguay: Comunicación 2024/159
Acceso
URL: http://192.168.2.226:8080/ValoresWeb/com.valoresweb.wloginvalores (esta es la interna Dynatech)
Aparecerá pantalla para acceso en la cual deberá tener usuario (un alias) y un password.
Dicho usuario deberá tener asociado una casilla, la cual debe estar asociado a una casilla o bien con el dominio de la empresa o casilla registrada en el BCU.
El usuario estará vinculado a una empresa y ésta a la empresa que le dé el soporte, de aquí en adelante denominada “Consultora”.
